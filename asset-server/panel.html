Inicjalizacja...

ℹ️ T2V — Wan2.1

  • Najlepsze wyniki z angielskim promptem
  • Format 480×832 = pionowy TikTok/Reels
  • Generacja ~5s klipu trwa ok. 3–6 min
  • VRAM ~8GB, zwalniany po zakończeniu

ℹ️ I2V — SVD-xt

  • Wgraj obraz lub kliknij 📸 I2V na filmie T2V
  • Pierwsze użycie: pobiera model ~10GB z HuggingFace
  • Generacja ~3.5s klipu trwa ok. 1–3 min
  • Ruch subtelny (1–80) / średni (127) / intensywny (200+)

ℹ️ I2V — Wan2.1 14B

  • Obraz + opis ruchu → wideo wysokiej jakości
  • Block offloading: ~4-6GB VRAM, eagleai.ai działa równolegle
  • 480P (~25min) / 720P (~40min) na 49 klatek
  • Prompt opisuje ruch i atmosferę, nie wygląd (to dyktuje obraz)
